AlphaGenomeR 0.99.0
-------------------

NEW FEATURES

    * Initial Bioconductor submission.
    * R interface to Google DeepMind AlphaGenome API via reticulate.
    * Support for 11 multimodal prediction tracks: RNA-seq, ATAC-seq,
      DNase-seq, CAGE, TF ChIP-seq, histone ChIP-seq, PRO-cap, splicing
      (sites, junctions, usage), and 3D genome contact maps.
    * Query genomic intervals up to 1 Mb with ontology-aware tissue
      specificity.
