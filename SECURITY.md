# Security Policy

## Supported Versions

Only the latest version of AlphaGenomeR is supported for security updates.

| Version | Supported          |
| ------- | ------------------ |
| 0.99.x  | :white_check_mark: |
| < 0.99  | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please report it by opening a draft security advisory in the [GitHub repository](https://github.com/BDB-Genomics/AlphaGenomeR/security/advisories/new) or contact the maintainers directly via the email listed in the `DESCRIPTION` file.

We aim to respond to all reports within 48 hours.
